Estructura de textos con etiquetas

Algunas veces vamos a querer darle formato a nuestros parrafos haciendo un "quiebre de linea" como en este ejemplo.

Lorem ipsum dolor sit amet consectetur adipisicing elit. <br> Provident itaque deserunt voluptatibus<br> est porro earum excepturi temporibus tempore<br> velit quaerat rem sint ducimus<br> doloribus, exercitationem quibusdam<br> repellat rerum aperiam eligendi!

Utilizando la etiqueta <br> el texto quedaria de este modo:

Lorem ipsum dolor sit amet consectetur adipisicing elit.
Provident itaque deserunt voluptatibus
est porro earum excepturi temporibus tempore
velit quaerat rem sint ducimus
doloribus, exercitationem quibusdam
repellat rerum aperiam eligendi!

Tambien podemos querer darle enfasis a los textos, usando una etiqueta llamada "emphasis" como en este ejemplo.

Lorem ipsum <em>dolor sit amet</em> consectetur adipisicing elit.

Utilizando la etiqueta <em></em> el texto quedaria de este modo:

Lorem ipsum dolor sit amet consectetur adipisicing elit.

Para definir un texto importante, nosotros usamos la etiqueta "strong" como en este ejemplo:

Lorem ipsum <strong>dolor sit amet</strong> consectetur adipisicing elit.

Utilizando la etiqueta <strong></strong> el texto quedaria de este modo:

Lorem ipsum dolor sit amet consectetur adipisicing elit.

El Elemento Head.

<head> es el contenedor de informacion acerca de la pagina web, tal como el titulo.
Y va antes de la etiqueta body.
